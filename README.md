# CK — Personal Portfolio

一个简洁、响应式的个人作品集网站，用于展示个人介绍、精选项目与联系方式。

**在线访问：[https://chi-ck.github.io/](https://chi-ck.github.io/)**

## 网站特点

- 响应式设计，适配桌面端和移动端
- 明亮 / 深色主题切换
- 精选项目展示区域
- 个人介绍与技能信息
- 联系方式及社交平台入口
- 纯 HTML、CSS 和 JavaScript，无需安装依赖
- 通过 GitHub Pages 自动发布

## 项目结构

```text
.
├── index.html    # 页面内容与结构
├── styles.css    # 页面样式与响应式布局
├── script.js     # 主题切换等交互
└── README.md     # 项目说明
```

## 修改内容

网站的文字、项目名称、邮箱和链接都位于 `index.html` 中。直接编辑并提交即可更新网站。

常用替换项：

- `你的名字` / `YOUR NAME`
- 个人简介和所在地
- 项目名称、说明与链接
- `hello@example.com`
- GitHub 及其他社交平台链接

样式、颜色和排版可以在 `styles.css` 中调整。

## 部署

本项目通过 GitHub Pages 发布。仓库设置为从 `main` 分支的根目录部署后，每次推送到 `main` 都会自动更新线上网站。

```bash
git add .
git commit -m "Update portfolio"
git push
```

部署通常会在提交后几分钟内完成。
