# 个人作品集网站模板

这是一个无需构建工具的静态网站，可以直接部署到 GitHub Pages。

## GitHub 网页端发布

1. 在 GitHub 新建一个公开仓库（建议命名为 `你的用户名.github.io`）。
2. 点击 **Add file → Upload files**，上传 `index.html`、`styles.css`、`script.js`。
3. 打开仓库 **Settings → Pages**，在 **Build and deployment** 中选择 **Deploy from a branch**。
4. 分支选择 `main`，目录选择 `/ (root)`，点击 **Save**。
5. 等待约 1–2 分钟，访问 `https://你的用户名.github.io/`。

## 需要替换的内容

在 `index.html` 中搜索并替换 `你的名字`、`YOUR / NAME`、项目名称、邮箱和 GitHub 链接即可。项目视觉目前使用 CSS 抽象图形，之后可以替换为真实项目截图。
